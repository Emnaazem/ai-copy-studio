# 🧠 AI Copy Studio

A simple AI-style web application built with HTML, CSS, and JavaScript.

It generates product descriptions based on tone and audience using simple smart logic (no backend).

---

## 🚀 Live Demo

👉 https://incomparable-tanuki-f2d492.netlify.app/

---

## ✨ Features

- Generate product descriptions instantly
- Choose tone:
  - Luxury ✨
  - Friendly 😊
  - Persuasive 🔥
- Target audience input
- Simple clean UI (mobile friendly)
- Fast and lightweight (no frameworks)

---

## 🛠️ Built With

- HTML
- CSS
- JavaScript (Vanilla)
- No backend required

---

## 💡 How it works

This project simulates AI behavior using JavaScript logic.
Based on user inputs, it generates dynamic marketing text.
